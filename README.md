# Isključenja Struje

Sistem automatiziranog i ranog obavještavanja planiranih isključenja struje.

## O projektu

Cijeli projekt je razvijen **<u>30. Novembra, 2025. godine</u>**, iz frustracije trajnih (dužih i kraćih) nestanaka
struje na potezu Previle-Tinja-Potpeć, i svih okolnih mjesta.

S obzirom da je cijeli projekt modeliran i izrađen za manje od jednog dana, ta činjenica otvara dosta prostora za
refactoring i unapređenja, ali ovaj MVP radi optimalno.

Deployment procedura nije navedena, ali bilo ko ko se snalazi u Node.js ekosistemu može veoma lako pokrenuti projekt
kod sebe.

Koristi Viber Channels API, fetch za HTTP layer, cheerio za HTML parsiranje, better-sqlite3 za perzistenciju.

Ovaj dokument će naknadno biti unaprijeđen kako projekt bude nadograđivan.
